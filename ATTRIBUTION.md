# AppForge

> Turn any website into a native desktop app — with a visual drag-and-drop interface.

AppForge is a GUI-powered app maker built on top of [Pake](https://github.com/tw93/Pake). It adds a visual interface, batch generation, profile management, and custom CSS/JS injection to Pake's powerful CLI.

## Features

- **Visual App Builder** — Drag-and-drop GUI with URL input, icon picker, and window size presets
- **Batch Generation** — Queue multiple URLs and build all apps in one run
- **Profile System** — Save and reload app configurations as reusable profiles
- **Custom CSS/JS Injection** — Paste custom styles or scripts to restyle pages before packaging
- **Global Hotkey** — `Ctrl+Shift+A` to toggle the app window
- **System Tray** — Minimize to tray with quick actions
